# Storage

This module introduced me to storage and key storage concepts, providing hands-on experience in understanding how storage and archiving controls work within modern systems and cloud environments

##Topics Covered

- Cloud storage
- Amazon EBS
- Instance Store
- Amazon EFS
- Amazon S3
- Amazon S3 Glacier
- AWS Storage Gateway

## Benefits / Key Takeaways

## Through this module, I learned how to:

- Understand different AWS storage services and when to use each one
- Create, attach, and manage Amazon EBS volumes for EC2 instances
- Work with Instance Store for high-performance, temporary storage
- Configure and use Amazon EFS for scalable file storage
- Upload, organise, and secure data in Amazon S3
- Store archived data cost-effectively using S3 Glacier
- Integrate on-premises environments with AWS using Storage Gateway
- Apply best practices for durability, availability, and data protection in the cloud
