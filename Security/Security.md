# Security
- This module introduced me to security and key security concepts, providing hands-on experience in understanding how security principles, policies and protective  controls work within modern systems and cloud environments
  
## Topics Covered
- Introduction to security
- Security lifecycle and prevention strategies
- Network hardening
- System hardening
- Data security
- AWS Identity and Access Management (IAM)
- AWS CloudTrail
- AWS Trusted Advisor
  
## Benefits / Key Takeaways
## Through this module, I learned how to:
- Apply basic security principles to protect systems and networks
- Harden networks and operating systems
- Implement data protection measures
- Manage users, roles and permissions using AWS IAM
- Monitor and audit activity using AWS CloudTrail
- Use AWS Trusted Advisor to identify security improvements
