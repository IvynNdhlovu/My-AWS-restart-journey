# Launching an EC2 Instance

- This practical lab introduced me to launching and managing an Amazon EC2 instance

# Lab Activities

## During the lab, we covered the following:

- Launch a web server with termination protection enabled
- Monitor your EC2 instance
- Modify the security group to allow HTTP access
- Resize the Amazon EC2 instance to scale
- Test termination protection
- Terminate the EC2 instance
- Steps Performed
- Access the AWS Management Console
- Launch EC2 instance and name it
- Choose an Amazon Machine Image (AMI)
- Select instance type
- Configure key pair
- Configure network settings
- Add storage
- Configure advanced details
- Launch the instance
- Monitor the instance
- Update the security group
- Resize the instance (change instance type and EBS volume)
- Test termination protection
  
# Benefits / Takeaways

## Through this lab, I learned how to:

- Access the AWS Management Console
- Launch and name an EC2 instance
- Monitor instance performance
- Update security groups to allow HTTP access
- Resize the instance type and EBS volume
- Test and manage termination protection
